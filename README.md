# Productivity Dashboard

A task management web application built with React, TypeScript, and Tailwind CSS, featuring real-time weather information.

## Features

- Add, complete, and delete tasks
- Search and filter tasks in real-time
- Weather widget powered by Open-Meteo API
- Dark / Light mode toggle
- Data persists with localStorage

## Tech Stack

- React + TypeScript
- Tailwind CSS
- Vite
- Open-Meteo API (no API key required)

## Getting Started

### Prerequisites
- Node.js 18+

### Installation

git clone https://github.com/GXYEKP/productivity-dashboard.git
cd productivity-dashboard
npm install
npm run dev

Open http://localhost:5173
